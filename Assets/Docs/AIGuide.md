# AI 활용 가이드

## AI를 무엇을 쓸 것인가?
  - 개인에게 맞는 AI Model 선택 하기.

## 지킬 것
  - **단순한 "해줘" 하지 말기**.
  - AI 질문 전 **5분 이상 고민**하기.
  - AI 가 **Code(Script)로 답변하지 않게 하기**.
      - 단, 예제 코드는 제외.
  - **구조는 먼저 설계 후 리뷰만 요청**.
      - 구조 설계가 매우 난해한 경우, 팀원들 혹은 강사님과 먼저 이야기 합시다.
      - 구조 내 상세 기능 구현시에 AI 의 힘을 빌린다.

## **매우 중요: AI 사용 문서**
  - 아래 흐름에 맞춰 작성
    - **이슈**; AI를 왜 사용하게 되었는지.
    - **Prompt**; AI에게 질문한 내용 혹은 접근 방법.
    - **결과**; AI가 내놓은 답변.
  - 정리를 AI가 하는 것은 상관 없음. 
    - 단, 위 흐름을 지켜야 하며 각 내용에 대하여 충분히 이해가 가능하도록 정리하기.
